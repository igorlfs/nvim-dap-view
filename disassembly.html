Disassembly View

The disassembly view is a custom view, built on top of nvim-dap-disasm. To enable it, you need to install nvim-dap-disasm. After installing the extension, all you have to do is adding it to your sections table:

return {
    sections = {
        -- ...
        "disassembly",
    },
}

    config = true,
}
