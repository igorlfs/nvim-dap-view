Hide Terminal

Some debug adapters don‚Äôt use the integrated terminal (console). To avoid having a completely useless window lying around, you can hide the terminal for them. To achieve that, add the following snippet to your nvim-dap-view setup:

return {
    windows = {
        terminal = {
            -- Use the actual names for the adapters you want to hide
            hide = { "go" }, -- `go` is known to not use the terminal.
        },
    },
}

Anchoring

One such scenario is when using the delve adapter for Go (more specifically, when using an attach request): the window with the terminal that launched dlv can act as if it was the nvim-dap-view‚Äôs terminal window.
